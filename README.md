# Donut App

A new Flutter project to demonstrate cross-platform mobile and desktop app development.

## Overview

Donut App is a starter application built using [Flutter](https://flutter.dev/). The project is designed as a learning resource and a foundation for building Flutter apps that target multiple platforms (Android, iOS, Linux, Windows, etc.).

## Features

- Simple authentication logic with login and signup methods (see `auth_repository.dart`)
- Cross-platform build setup for Android, Windows, and Linux
- Easily extensible architecture for new features

## Getting Started

A Flutter Application for a Donut Shop ( Half Build) with Login / Signup using Bloc Pattern.

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- Dart SDK (included with Flutter)
- Platform-specific requirements (e.g., Android Studio, Visual Studio, etc.)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Udai-Senevirathne/Donut-App.git
   cd Donut-App/donut_app
   ```

2. Install Flutter dependencies:
   ```bash
   flutter pub get
   ```
## Project Structure

- `lib/` - Main Dart code for the application
  - `bloc/auth_repository.dart` - Contains simple authentication logic (login and signup)
- `android/`, `windows/`, `linux/` - Platform-specific build configuration files
- `test/` - Unit and widget tests
